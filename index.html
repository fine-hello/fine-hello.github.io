我的个人主页（实验报告）

热爱祖国，积极向上——以网页展示路径引用与页面设计

实验目的

  • 了解 GitHub 与 GitHub Pages 的基本使用与托管原理。
  • 掌握使用 HTML/CSS 进行基础页面排版与设计。
  • 理解 Web 项目中相对路径与绝对路径的概念与应用场景。

图片展示（满足实验要求）

下面展示两张图片：左图使用绝对路径引用，右图使用相对路径引用（相对路径基于本仓库根目录）。

绝对路径引用

这张图片使用绝对 URL（示例为 raw.githubusercontent.com 的绝对链接），用来表达我对祖国的热爱与自豪。

相对路径引用

这张图片使用相对路径（data/img.png），来自仓库内的本地资源，表达积极向上、奋发进取的精神。

实验总结（示例）

在本次实验中，我通过搭建 GitHub Pages 的个人主页，实践了 HTML 与 CSS 的页面布局，并验证了绝对路径与相对路径在 Web 项目中的不同用途。绝对路径便于引用外部网络资源或跨站点共享文件；相对路径更适合项目内部资源管理与发布。页面内容表达了我对祖国的热爱与积极进取的态度，可作为实验报告提交内容之一。
